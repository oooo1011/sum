# 数字组合计算器

一个Python GUI应用，用于在给定数字列表中找出总和等于目标值的组合。

## 功能特性

- 支持导入数字列表（txt/csv格式）
- 可设置目标值
- 可选择查找第一个组合或所有组合
- 优化算法减少内存占用
- 实时计算进度显示

## 使用说明

1. 安装依赖: `pip install -r requirements.txt`
2. 运行程序: `python main.py`
3. 点击"导入数字"按钮选择数字文件
4. 设置目标值
5. 选择是否查找所有组合
6. 点击"计算组合"按钮

## 技术栈

- Python 3.x
- Tkinter (GUI)
- 动态规划算法
